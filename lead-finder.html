📖 Setup Guide
STEP 1 — Free API Key
1. Open app.hasdata.com
2. Sign up with Google — no card needed
3. Profile → API Key → copy it
4. Tap ⚙️ above → paste the key
→ 1,000 free credits = ~200 searches

STEP 2 — Deploy to Netlify (fixes API)
1. Save this file → go to netlify.com
2. Sign up free → upload this HTML file
3. Use the Netlify live URL always
→ Required: local files block API calls
